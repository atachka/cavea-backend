## How to run

1. Add .env file to the root of the project, for example:

PORT=4000  
 DB_HOST=localhost  
 DB_PORT=5432  
 DB_NAME=postgres  
 DB_USER=postgres  
 DB_PASSWORD=admin

2. run npm install

3. run npm start

## Scripts

1. run project once (npm start)

2. npm run "script name"
